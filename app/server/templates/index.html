OgramCloud Beta

Freely Transform your TELEGRAM ACCOUNT into a CLOUD-STORAGE-SERVER without registration/tracking !

OgramCoud is an experimental Open Source project made by Sanix-darker - Contributors list.

How it works?

  1. Get your chatID from Ogram-Telegram-bot
  2. Use it to save files

    3. Make sure to always save your files key ('file-key')

  3. To get your file back, then copy the file-key and generate the download link

    4. All files are deleted all 24h after download-link generated.

⚠︎ Because it's an experimental project, for now you can just store 100 Mo of files.
