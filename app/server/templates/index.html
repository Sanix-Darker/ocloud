OgramCloud Beta

Freely Transform your TELEGRAM ACCOUNT into a CLOUD-STORAGE-SERVER without registration/tracking, NO LIMIT IN STORAGE.

OgramCoud is an Open Source test project made for FUN by Sanix-darker, -   Contributors list

How it works?

  1. Get your chatID from Ogram-Telegram-bot.
  2. Use it to save your files, you will get a generated file-key !

    3. Your file is on Telegram-Servers now, using your account, make sure to save your files's keys('file-key'), you can share the download link too.

  3. To regenerate your file, copy the file-key and generate the download link.

    4. All files are deleted from OgramCloud Tempory Storage, each 24H after their download-link have been generated.

⚠︎ Because it's on BETA version in a small server, for now you can just store only file < 100 Mb per upload.
